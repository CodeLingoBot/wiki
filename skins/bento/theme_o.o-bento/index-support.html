Navigation Structure

The new navigation structure is Task Oriented. Informations should be where a user from our target groups expects them. This means that informations are linked on several places.

We broke down the main audience of opensuse.org to:

  • Users
  • Community
  • Developer or Contributor

For more details go to "User Qualification Classes" on the main project page.

Based on this idea we have four main categories or clusters:

  1. Downloads
  2. Support (or Help)
  3. Community
  4. Development

Cluster Content

Downloads

Here users should find everything what we offer for download and cross links to support pages:

  • Distribution download (software.opensuse.org)
  • Developer Releases (software.opensuse.org/developer/)
  • Software-Package download (software.opensuse.org/search)
  • Links to related 3rd Party Websites

Support (or Help)

The Support cluster contains everything to help a user with his problem:

  • openSUSE Wiki
  • Simple tutorials (HOWTO-style, Screencasts)
  • Forums
  • Related mailing List (ML)
    • simple registration
    • ML-Archive
  • List of related IRC channels

Community

All the Community related things with a social flavour are wrapped here:

  • Blogs (e.g. news.opensuse.org, lizards.opensuse.org, spotlight.opensuse.org)
  • "User Account/Whatever Management" (users.opensuse.org)
  • Upcoming events
  • Merchandising (shop.opensuse.org)
  • [Maybe] Feature Voting System
  • Forums
  • All mailing Lists
  • List of related IRC channels
  • Related Wiki categories
  • ...

Development

In this cluster there should be everything a contributor/developer needs to contribute:

  • Developer Center (Contributor specific documentation)
  • Related mailing Lists
  • List of related IRC channels
  • Feature Overview (features.opensuse.org)
  • openSUSE Build Service (build.opensuse.org)
  • Related Wiki categories

Favorites - Function

The global navigation will also contain a "Favorites"-Button, which could by default contain the most views pages. A logged in User can edit his favorites and create on this way a custom navigation item.
